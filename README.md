# CodevaMP

Plataforma creativa-tecnica con foco en producto real, colaboracion y publicacion.

## Arranque local

```bash
npm install
npm run dev
```

App local por defecto:
- `http://localhost:3000`

## Rutas principales

- `/` Home del ecosistema.
- `/dev` Unidad de desarrollo (feed, publicacion y gestion de proyectos).
- `/dev/studio` Workspace tecnico.
- `/visuales` Comunidad visual.
- `/dashboard` Vista ejecutiva por subempresa.
- `/estrategia` Plan de propuesta de valor + roadmap de impacto.

## Documentos de estrategia

- `docs/strategy/value-proposition-2026.md`
- `docs/strategy/bcorp-roadmap-2026.md`

## Comandos utiles

- `npm run dev`: desarrollo con Turbopack.
- `npm run build`: build de produccion.
- `npm run start`: correr build de produccion.
- `npm run lint`: validacion de codigo.
